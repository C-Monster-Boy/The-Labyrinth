This is the first prototype for the game : THE LABYRINTH



#Controls:
- WASD to move
- Use Mouse to look around
- Left Mouse Button to drop a breadcrumb
- Q to use dissaper power
- SpaceBar to use Birds Eye View Power

# How to Play
The player is spawned in the maze. The goal is to NAVIGATE THE MAZE to reach the PORTAL.
But beware... there is a deadly bot on the hunt.
